Corresponds to Algorithm 5 (Partition Strategy) in Report 10 [H_A, P, z, Q_tilde, R_tilde, indicator] = partition_H(A, z_B, m, n) permutes and partitions H_A so that the submatrices H_i are full-column rank

Inputs: A - m-by-n real matrix z_B - n-dimensional integer vector m - integer scalar n - integer scalar

Outputs: P - n-by-n real matrix, permutation such that H_A*P=A z - n-dimensional integer vector (z_B permuted to correspond to H_A) Q_tilde - m-by-n real matrix (Q factors) R_tilde - m-by-n real matrix (R factors) indicator - 2-by-q integer matrix (indicates submatrices of H_A)
